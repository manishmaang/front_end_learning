CSS POSITION PROPERTY

  • static : default position hota hai aur isme element apne normal flow me rehta hai aur top right bottom left properties kaam nhi krti hai and z-index kaam nhi krta hai.
  • relative : relative position ka matlab hai ki element apne normal position se kitna move hoga and iske liye hum top right bottom left properties ka use krte hai and ye properties element ke normal position se hi calculate hoti hai. and ye sirf usi element ko move krta hai and uske siblings ko move nhi krta hai. (and if ye zyada move ho jaye to siblinds ke upar bhi chala jata hai but it won't affect the siblings position)
  • absolute : absolute position ka matlab hai ki element apne nearest positioned ancestor se kitna move hoga and iske liye hum top right bottom left properties ka use krte hai and ye properties element ke nearest positioned ancestor se hi calculate hoti hai. (and if there is no positioned ancestor then it will be relative to the initial containing block which is usually the html element) ;
  • agr parent ki position default set hai but mene parent me transform, filter or perspective property use kiya hai to bhi child element ka position absolute set karne par vo parent ke relative ho jayega instead of root html element.
  • When you set .box3 to position: absolute, it is taken out of the normal flow. That means it no longer pushes or reserves space for itself — it just “floats” relative to its nearest positioned ancestor (or the element if none exists).
  • fixed : fixed position ka matlab hai ki element viewport se kitna move hoga and iske liye hum top right bottom left properties ka use krte hai and ye properties element ke viewport se hi calculate hoti hai. (and it will always stay in the same position even when the page is scrolled)
  • sticky : sticky position ka matlab hai ki element apne normal position se kitna move hoga and iske liye hum top right bottom left properties ka use krte hai and ye properties element ke normal position se hi
