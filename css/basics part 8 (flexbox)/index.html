FLEX BOX

  • display: flex; it should be applied to the parent container for flexbox behavior
  • flex-direction: row | column; controls the direction of the flex items
  • there are 2 axis: main axis and cross axis and justify content and align items works on them
  • justify content controls the alignment of flex items along the main axis and align items controls the alignment of flex items along the cross axis
  • if there are more than one line of flex items, align-content controls the alignment of the lines
  • align items controls the alignment of flex items along the cross axis and when there are multiple lines we use align-content.
  • when flex direction is row, the main axis is horizontal and the cross axis is vertical and when flex direction is column, the main axis is vertical and the cross axis is horizontal
  • order property can be used to change the order of flex items and by default sbka order 0 hota hai jitna km order vo sbse pehle
  • almost all of the properties are applid in the parent container so don't mix the parent's and child's properties
  • flex grow, flex shrink, and flex basis control how flex items expand and shrink
