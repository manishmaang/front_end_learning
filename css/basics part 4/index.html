I'm a none element
display property is used to specify the display behavior of an element and it has 4 main values :
  • block : will take up the full width of its parent container
  • inline : will only take up as much width as needed (can't set the width, top margin and top padding)
  • inline-block : will only take up as much width as needed but can have a width and height(top margin and padding)
  • none : will hide the element
  • visible : agr visible kiya hai to element ke sth sth uski space bhi chle jayegi but agr hum chahte hai sirf element hidden rhe but uski space rahe to hum visible property use kr skte hai.
