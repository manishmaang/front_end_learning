All About CSS Specificity

  1. Specificity : is a set of rules that browsers use to determine which CSS rule applies when there are multiple rules that could apply to the same element. It is calculated based on the types of selectors used in the CSS rule. The more specific a selector is, the higher its specificity value, and the more likely it is to be applied to an element.

All About CSS Units

  1. px => pixels => absolute unit => 1px is equal to 1/96th of an inch. It is a fixed unit and does not change based on the screen size or resolution. It is commonly used for precise control over the layout and design of a webpage.
  2. vw => viewport width => relative unit => 1vw is equal to 1% of the width of the viewport. It is a flexible unit that adjusts based on the size of the viewport, making it useful for responsive design.

    ye screen ke width ke hisab se adjust hota hai and element ke width ko control karta hai

  3. vh => viewport height => relative unit => 1vh is equal to 1% of the height of the viewport. It is also a flexible unit that adjusts based on the size of the viewport, making it useful for responsive design.

    ye screen ke height ke hisab se adjust hota hai and element ke height ko control karta hai

  4. em => relative to the font-size of the element => agr 1 div ke andr ek p element hia and div ka font size 16 hai to by default paragraph ka bhi font size 16px hi hoga kyuki vo div se inherit karega but agr hum paragraph ke font size ko 2em kar dete hai to paragraph ka font size 32px ho jayega kyuki 2em ka matlab hai 2 times the font size of the parent element (div) which is 16px. and rest of the li or article or normal fonts ka size 16px hi hoga kyuki unka font size em me nhi diya gaya hai to vo div se inherit karenge jo ki 16px hai.
  5. rem => relative to the font-size of the root element (html) => 1rem is equal to the font size of the root element. It is a flexible unit that allows for consistent sizing across different elements and is often used for responsive design.

    ye root element ke font size ke hisab se adjust hota hai and element ke size ko control karta hai

  6. vmin & vmax => viewport minimum and maximum => 1vmin is equal to 1% of the smaller dimension of the viewport (either width or height), while 1vmax is equal to 1% of the larger dimension. These are flexible units that adjust based on the size of the viewport, making them useful for responsive design.

    height ya width jo choti hai uski size lelega. and vice versa for vmax (jo bda hai uski size lelega)

  7. % => percentage => relative unit => 1% is equal to 1/100th of the parent element's size. It is a flexible unit that adjusts based on the size of the parent element, making it useful for responsive design.

    ye parent element ke hisab se adjust hota hai and element ke size ko control karta hai
