semantic tags

  • semantic tags are the tags which have some meaning or semantic value to them like header, nav, main, article, section, aside, footer etc etc
  • these tags are used to structure our webpage in a better way and also helps in seo and accessibility of our webpage
  • these helps browser engine and web crawlers to understand the structure of our webpage
  • styling still needs to be done with the help of css these are just structural elements and hold no default property.
